라이브러리(모듈)
- 라이브러리(library) -> 도서관
- 라이브러리(library)는 모듈(module)이라고도 불리며, 자주 쓰는 함수,
    변수들을 모아 놓은 파이썬 파일이다.
- 자주 사용하는 기능을 모듈화, 패키지화 하여 만들어 둔 것

- 표준 라이브러리
    Python 에서 기본으로 제공하는 라이브러리

- 외부 라이브러리
    개발자가 필요에 의해 개발한 패키지와 모듈의 집합

학습목표
- 모듈의 개념에 대해서 이해한다.
- Numpy 라이브러리를 활용할 수 있다.

모듈(module)
- 필요한 코드를 재사용하기 위해 변수나 함수 또는 클래스를 모아 놓은 파일
- 다른 파이썬 프로그램에서 불러와 사용할 수 있게끔 만든 파이썬 파일
- 모듈은 다른 사람이 만들어 놓은 모듈을 사용할 수도 있고 직접 만들수도 있다.

numpy  pandas  matplotlib

데이터 분석에 특화된 모듈(라이브러리)

Numpy - 고성능 과학계산을 위한 데이터분석 라이브러리
Pandas - 행과 열로 구성된 표 형식의 데이터를 지원하는 라이브러리
Matplotlib - 2D 그래프로 시각화가 가능한 라이브러리

numpy.ndarray 클래스
- 동일한 저료형을 가지는 값들이 배열 형태로 존재함.
- N차원 형태로 구성이 가능하다.
- 각 값들은 양의 정수로 색인(index)이 부여되어 있다.
- ndarray를 줄여서 array로 표현한다.

Pandas
직관적으로 보기위해, 쉽고 편하게 보기위해
pandas 에서 제공하는 데이터구조
데이터 조작 및 분석을 위한 라이브러리
- Series Class : 1차원
    : 인덱스(index) + 값(value)
- DataFrame Class : 2차원
    : 행과 열을 가지는 표와 같은 형태
    : 서로 다른 종류의 자료형을 저장할수 있다.

Pandas 모듈 import
- import pandas as pd
    : pandas모듈(라이브러리)를 import하고 앞으로 pd라는 이름으로 부르겠다.
