F A T : The Role of Explainable AI

Fairness : AI에 의한 의사결정 과정 및 결과는 모든 사람에게 공정해야 한다.

Accountabillty : 인간 개발자는 AI가 야기하는 모든 부정적인 효과를 통제할 의지와 역량을 지녀야 한다.

Transparency : AI에 의한 의사결정 과정 및 결과에 누구든 접근 가능해야 하며, 누구든 이해할 수 있어야 한다.

1. 확률과 통계

 경우의 수
- 경우의 수 : 미래에 발생할 수 있는 사건의 가짓수가 n개라고 할 때, 이를 'n개의 경우의 수'로 정의

- 합의 법칙(or) : 경우의 수를 구해야 하는 상황이 서로 영향을 주거나, 
그 구조가 닳지 않은 경우, 이를 합쳐서 경우의 수를 세는 방식

- 곱의 법칙(and) : 경우의 수를 구해야 하는 상황이 서로 영향을 주지 않거나, 
그 구조가 닳은 경우, 이를 곱해서 경우의 수를 세는 방식

1. 동전을 4번 던질 때 일어날 수 있는 모든 경우의 수는?
2 x 2 x 2 x 2 = 16
2. 두 사람이 가위바위보를 할 때 일어날 수 있는 모든 경우의 수는?
3 x 3 = 9
3. 6명의 학생을 한 줄로 세우고자 할 때, 일어날 수 있는 모든 경우의 수는?
6! = 6 x 5 x 4 x 3 x 2 x 1 = 720
4. 6명의 학생 중에서 3명의 학생을 무작위로 뽑고자 할 때, 뽑힌 구성원의 모든 경우의 수는?
6c3 = 20
5. 6명의 학생 중에서 3명의 학생을 뽑고, 그 학생들을 한 줄로 세우고자 할 때 모든 경우의 수는?
6p3 = 120

2. 회귀분석
- 독립변수 : x , 종속변수 : y
- mse(mean-squared-error) : 평균제곱오차
- 머신러닝의 회귀 vs 통계의 회귀(x값과 y값의 상간관계)

- 척도의 종류
1. 명목척도(Norminal Scale)
2. 서열척도(Ordinal Scale)(덧셈 뺄셈 불가능)
3. 등간척도(Interval Scale)(덧셈 뺄셈 가능)
4. 비율척도(Ratio Scale)(곱셈까지 가능)
5. 범주형척도(변수의 수를 셀수있음)1개 2개 3개
6. 연속형척도(변수 사이의 다른 값이 존재할수있을때)15도 16도

- 회귀에서 봐야 하는 3가지 조건
1. 독립변수가 종속변수를 얼마나 잘 설명하는가
2. 독립변수가 종속변수를 정말로 잘 설명하는가
3. 똑같은 분석을 한 번 더 해도 비슷한 결과가 나오는가( = 이번 분석의 결과가 필연적인가)

회귀 기본용어
y(종속변수) = β(베타,회귀 계수) * x(독립변수) + a(y절편) + ε(오차)

1. 독립변수가 종속 변수를 얼마나 잘 설명하였는가
y(종속변수) = β(회귀계수) * x(독립변수) + a(y절편) + ε(오차)

1. 회귀 계수를 본다.
- > β = 2라는 것은, 독립 변수가 1만큼 커지면 종속 변수는 2만큼 커진다는 뜻
2. 표준화 계수를 본다.
-> 독립 변수와 종속 변수의 단위가 다를수도 있잖아?
    이 경우에는 독립 변수와 종속 변수를 표준화 해서 본다.
3. R²(설명력)를 본다.
R² = 편차 / 편차-오차

- 편차 = 실제 값 - 예측값의 평균
- 오차 = 실제 값 - 예측값    

2. 독립변수가 종속변수를 정말로 잘 설명하는가
y = βx + ε

3. 똑같은 분석을 한번 더 해도 비슷한 결과가 나오는가?
    1. 표준 오차를 본다.
    -> 표준 오차 = 표준 평균의 표준 편차
    2. t값과 p값을 본다.

- 회귀의 전제조건

선형성 : 회귀의 추세가 선형성을 띄는가?

- 표본 검증(Sampling)

1. 확률적 추출법
    - 무작위 추출법(Random Sampling) : 모집단 내에서 임의의 표본을 추출
    (ex) 출구조사 시 선거를 하고 나오는 사람들을 무작위로 설문하는 경우
    - 체계적 추출(System Sampling) : 특정 비율로 표본을 추출
    (ex) 출구조사 시 선거를 하고 나오는 10명마다 1명씩 설문하는 경우
    - 충화 추출법(Stratified Sampling) : 특정 표본집단의 비율을 반영
    (ex) 서울 인구는 1천만명, 전라도 인구는 500만명 이므로 서울에서 100명, 전라도에서 50명을 설문하는 경우
2. 비확률적 추출법
    - 편의 추출(Convenience Sampling) : 내 마음대로 표본을 추출
    (ex) 출구조사 시 선거를 하고 나오는 사람들 중 내가 마음에 드는 사람만 설문하는 경우
    - 연쇄 추출(Snowball Sampling) : 특수한 특성을 가진 사람들을 표본으로 선정
    (ex) 정치 전문가를 찾아가 설문하는 경우
    - 고의 추출(Purposeful Sampling) : 특저안 이유로 일부 표본을 측정
    (ex) 출구조사 시 선거 부스 내에서 1분 이상 머무른 사람에 한해서 설문하는 경우
3. 카이제곱검정(Chi-squared Goodness to Fit Test)
    카이제곱검정 = 기댓값 분에 (관측값 - 기댓값)²

- 타당성 검증(Validity)

수렴타당성(Convergent Validity)
    독립변수들이 종속변수를 일관적으로 설명하고 있는가?
    - AVE(Average Variance Extracted) : 특정 하나의 항목의 분산을 측정
    ( 0 < x < 1 이며, 보통 기준은 0.5 이상)
    -크론바하 알파(Cronbach's Alpha) : 특정 하나의 항목의 분산과, 모든 항목의 평균 분산을 비교
    ( 0 < x < 1 이며, 보통 기준은 0.7 이상)
    - CR(Composite Reliability) : 특정 하나의 항목의 오차와, 모든 항목의 오차를 비교
    ( 0 < x < 1 이며, 보통 기준은 0.7 이상)
구별타당성(Discriminant Validity)
    독립변수 간의 상간관계가 없는가?
    - VIF(Variance Inflation Factor) : 설명력의 역수로, 특정 독립변수가 빠졌음에도 실명력이 크게 줄지 않는 경우
    ( 0 < X 이며, 보통 기준은 10 미만)
    - 상관계수 행렬 (Correlation Matrix) : 변수 A가 커지는 조건에서, 변수 B도 같이 커지는 경우
    ( 0 < x < 1 이며, 보통 기준은 0.85 미만)
    - 공분산 행렬(Covariance Matrix) : 다중공선성이 발생하는 경우
    (0 < x < 1 이며, 보통 기준은 0.85 미만)

- 독립성 검증
하위변수와 혼란 변수 y = βx + ε

    혼란변수 : A와 B에 모두 영향을 주어 마치 A와 B의 상관관계를 과장하는 변수

고급 회귀
1. 다항회귀
    - 추세가 직선이 아니고, 다른 추세를 사용하는 경우
    - 기본형은 y = βxn + a + ε
    - 경우에 따라서 정규 분포, 지수/로그인함수, 삼각함수를 다르는 경우도 있음
2. 매개 변수
    - 독립변수가 종속변수에 직접 영향을 미치지는 않지만, 다른 변수에 의해서 간접적으로 영향을 미칠 수 있는 경우
    - 가령, 기온이 오르는 것 자체는 다이어트 의지에 직접적인 영향이 없지만, 기온이 올라서
        옷이 얇아진다고 생각하며느 다이어트 의지가 불탈 수 있음
    - 기본형은 y = (β₂xn + a₂ + ε₂) + a₁ + ε₁
    - 풀이랄 것은 없고, y = f(x)일 때, 매개변수가 있는 경우 합성함수 y=f(f(x))의 꼴
3. 조절변수
    - 종속변수에 직접 영향을 미치지는 않지만, 그 값에 따라 독립변수가 종속변수에 미치는 영향이 달라지는 경우
    - 예를 들어, 성별은 성장속도를 단독으로 설명하지 못하지만,
        여성은 평균 만 11세 - 만 16세가 성장기인 반면, 남성은 평균 만 13세 - 만 18세이므로, 나이와 성장 속도의 관계에 관여한다.
    - 기본형은 y = β₁x + β₂m + β₃mx + a + ε(m은 "주효과항", mx는 "상호작용항"이라고 부른다.)
