군집분석 = 클러스터링

K-Means
데이터와 군집의 중심점의 거리를 재서, 그 거리가 가장 가까운 군집에
속한다고 가정하는 방법

분석순서
1) K개의 중심점을 무작위로 정한다.
2) 중심점과 거리를 재서 가장 가까운 군집에 데이터를 할당한다.
3) 군집에 속한 멤버들의 평균을 내서 중심점을 다시 정한다.
4) 중심점에 변화가 없을 때 까지 반복한다.

계층적 군집분석(hierarchical clustering)
반복해서 데이터를 뭉치거나 쪼개는 방식으로 군집을 만드는 방법

분석 순서
1) 하나의 데이터를 하나의 군집으로 본다.
2) 모든 군집들 사이의 거리를 잰다.
3) 거리가 가장 가까운 두 군집을 하나의 군집으로 합친다.
4) 원하는 갯수의 군집이 될 때까지 반복한다.

군집 간 거리를 재는 방법
- Single : 두 군집에서 가장 가까운 데이터들의 거리를 잰다.
- Complete : 두 군집에서 최대 거리가 가장 짧은 데이터의 거리를 잰다.
- Average : 모든 데이터들 사이의 거리를 평균을 낸다.
- Centroids : 군집의 중심과 중심의 거리를 잰다.
- Ward : 두 군집을 합쳤을 때 군집 내 거리 분산의 변화를 거리의 척도로 삼는다.

DBSCAN
밀도 기반의 군집 알고리즘
최초의 임의의 점 하나로부터 점점 퍼져나가는 밀도에 
기반한 군집화 방식을 사용

분석 순서
1.임의의 점 p를 설정하고, p를 포함하여 주어진 클러스터의 반경 안에 포함되어
  있는 데이터 포인트들의 갯수를 센다.
2.반경 안에 minPts 개 이상의 데이터들이 포함되어 있으면 해당 점p를
  corepoint로 간주하고 반경 이내에 포함된 데이터들을 하나의 군집으로 묶는다.
3.반경 안에 minPts 개 미만의 데이터들이 포함되면 넘긴다.
4.1-3을 반복하는데 만약 새로운 점 p가 core point가 되고 이 점이 기존
   군집에 속한다면, 두 개의 클러스터는 하나의 군집으로 묶어준다.
5.모든 점에 대해서 군집 과정을 끝냈을 때, 어떤 점을 중심으로 하더라도
    군집에 속하지 못하는 데이터가 있다면 noise point로 간주한다.

용어
- Epsilon : 군집의 반경
- minPts : 군집을 이루는 데이터의 최소 개수(15)
- core point : 반경 epsilon 내에 minPts 개 이상의 점이 존재하는 중심점
- border point : 군집의 중심이 되지는 못하지만, 군집에 속하는 점
- Noise point : 군집에 속하지 못한 점
