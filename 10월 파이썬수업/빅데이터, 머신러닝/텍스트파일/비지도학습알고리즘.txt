어떤 구조를 가정하고, 데이터를 그 구조에 맞추어 설명하는 방법

정답 값이 주어지지 않은 입력 데이터들을 분석하여
공통점이 있는 데이터끼리 묶어 그룹핑하는 방식

비지도 학습의 특징
- 정답이 없기 때문에 알고리즘이 유용한 것을 학습했는지 평가하기가 어려움
- 비지도 학습 알고리즘을 EDA, 전처리 단계에서 활용하기도 함 (x_train) -> 비지도
- 스케일 조정 메서드도 지도 정보(supervised information)을 사용하지
  않으므로 비지도 방식.

비지도 학습의 종류
- 차원 축소 : 원래 데이터보다 적은 개수의 차원으로 데이터를 표현하는 것
- 군집 분석 : 데이터를 비슷한 것들끼리 일정한 무리 또는 군집으로 나누는 것

차원 축소(Dimensionality Reduction)
차원 축소를 하는 이류
- 차원이 증가하면 그 차원을 표현하기 위한
  데이터 양이 기하급수적으로 증가 ("차원의 저주")

차원 축소를 하는 이유
1) 복잡한 데이터를 2차원으로 시각화해서 나타냄
2) 변수들 사이의 공통적인 정보만 남겨 점수화 시킴
3) 차원을 줄여 차원의 저주를 피하고 지도학습의 성능을 높임

차원 축소의 종류
PCA, t-SNE, NMF

PCA의 적용
- 잠재의미분석(latent semantic analysis) : PCA를 텍스트 데이터에 적용한 것
- 고유이미지(eigenimage) : PCA를 이미지 데이터에 적용한것

PCA 분석 분서
1) 분산이 가장 큰 방향(벡터)를 찾는다.
2) 첫번째 단계에서 찾은 벡터와 직교하면서 가장 큰 정보를 담은 벡터를
   찾음(2차원에서는 2개, n차원에서는 n개)
3) 주성분을 구한 후 원래 데이터의 특성 공간에서 표현하기 위해 다시
   평균을 더한다.

PCA의 특징
- 데이터 분산에 대한 정보를 유지하려 함
- 내제된 정보 손실이 적음
- 선형 데이터에 적합함

PCA의 장단점
- 데이터 분포의 분산을 최대로 유지한 상태로 차원 축소를 하기때문에
  정확하고 간결하게 데이터 표현이 가능함
- 최대 분산 방향이 feature의 구분을 좋게 만들어준다는 보장은 없음

t-SNE(t-Distributed Stochastic Neighbor Embedding)
데이터 포인트 사이의 거리를 가장 잘 보존하는
2차원 표현을 찾는 것

t-SNE 분석 순서
1) 각 데이터 포인트를 2차원에 무작위로 표현한다.
2) 원본 특성 공간에서 가까운 포인트는 가깝게, 멀리 떨어진 포인트는
   멀어지게 만듦

t-SNE의 특징
- 비선형 데이터를 시각화하는데 적합함
- 상대적 거리 정보를 시각화하는데 유리함
- 시각화에 유용함

t-SNE의 장단점
- 차원 축소 전 후 데이터의 상대적 거리를 유지하기 때문에 시각화에
  유용한 알고리즘
- 물리적 의미를 무시해서 시각화 이외의 다른 분석 용도로 사용하기는
  어려움
